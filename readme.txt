Gifts 4 You
A simple web application that allows users to create events, share gift wishlists, and anonymously check off items they plan to purchase. 
Perfect for organizing events like baby showers, birthdays, and other special occasions where you want to receive gifts you actually need and love!

Features
User Authentication: Login and register screens to securely access your account.

Home Screen:
View events on a calendar.
See a list of people you may know 
and posts from friends.

Calendar Page:
View the calendar and all the events from your friends.
Create and manage events for selected dates.

Gift List Page:
See event details and the wishlist of the event creator.
Anonymously check off items you plan to buy.

Profile Page:
Add a profile picture.
See your personal posts, events, and followers.
